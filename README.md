# 🤖 CampusMate AI Chatbot

A modern AI chatbot developed using Python and Streamlit.  
This chatbot provides interactive responses with a stylish user interface and works directly in the browser.

---

# 🚀 Features

- Modern Dark UI
- AI Chat Responses
- Chat Bubble Design
- Responsive Layout
- Browser-Based Application
- Streamlit Integration
- Multiple Question Handling
- Fast and Lightweight

---

# 🛠️ Technologies Used

- Python
- Streamlit
- HTML/CSS Styling
- Random Module
- Datetime Module

```bash

```

## Step 2: Run the Application

```bash
streamlit run app.py
```
